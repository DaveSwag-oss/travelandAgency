## Train Booking Web App

This project is a simple web application for booking train journeys. It allows users to search for routes, view schedules, compare options, and simulate ticket booking through a clean and user-friendly interface.

### Technologies Used

* **HTML** for structuring the web pages
* **CSS** for styling and layout, including responsive design
* **JavaScript** for handling user interactions, dynamic content, and basic logic

### Features

* Route search functionality
* Display of train schedules and options
* Interactive UI with dynamic updates
* Responsive design for desktop and mobile devices

### Purpose

The goal of this project is to demonstrate the basics of front-end web development by building a functional and visually clear booking interface. It can be used as a starting point for more advanced travel or booking systems.
